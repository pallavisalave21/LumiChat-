# 💖 LumiChat – Aesthetic AI Chatbot

LumiChat is a modern, blush-pink themed AI chatbot designed with a clean and user-friendly interface. Built using HTML, CSS, and JavaScript, it integrates with the Gemini API to generate real-time responses and includes a fallback system for uninterrupted user experience.

## ✨ Features
- 💬 Real-time chat interface
- 🎨 Blush pink aesthetic UI with glassmorphism design
- 🤖 Gemini API integration for AI responses
- 🔁 Fallback replies when API is unavailable
- ⌨️ Send messages using Enter key
- 📱 Responsive and smooth UI animations

## 🛠️ Tech Stack
- HTML
- CSS (Glassmorphism + Gradient UI)
- JavaScript (Vanilla JS)
- Gemini API

## 🚀 How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Enter your Gemini API key in the script file
4. Start chatting!

## ⚠️ Note
If the API quota is exceeded, the chatbot will automatically switch to fallback responses.
